A modern block-puzzle game built with **React, Next.js, and HTML Canvas**.  
Drag and drop shapes onto the grid, clear lines, build combos, and survive as long as you can.

This project is a work in progress and is actively being refined and expanded.

---

## Game Modes

### Classic Mode
- No timer
- Score by clearing rows and columns
- Game ends when no shapes can fit on the board

### Timed Mode
- Play against the clock
- Score as much as possible before time runs out

### Endless Mode
- Continuous gameplay
- Difficulty increases over time

### Challenge Mode
- Complete objectives instead of chasing a score
- Clear specific shapes (stars, squares, triangles, etc.)
- Game ends when all targets are completed or no moves remain

> Some modes are still being refined as development continues.

---

## Features

- Drag-and-drop shape placement
- Ghost preview with valid/invalid placement feedback
- Smooth canvas animations and particle effects
- Combo system and score multipliers
- Multiple game modes
- Sound effects for placement, clearing, and errors
- Responsive and minimal UI
